# Models Documentation

## Bill

- Bill Id
- Number of Items
- Total Cost
- Order Date
- Status

## BillItem

- ItemId
- Name
- Price
- Total
- Amount
- Bill Id - links to bill (many to one)
- Menu Item Id - links to the menu item (one to one)

- Name
- Price
